A modern CSS normalization library for 2026. Targets evergreen browsers with CSS custom properties, dark mode, accessibility, and motion preferences built-in.

Features

🎨 CSS Custom Properties

All values exposed as --fn-* variables for easy theming

🌙 Auto Dark Mode

Responds to prefers-color-scheme automatically

♿ Accessibility

Focus-visible, sr-only, and skip-link utilities included

🏃 Reduced Motion

Disables all transitions for users who prefer it

🖨️ Print Styles

Clean, readable print layout with URL expansion

📜 Scrollbar

Consistent scrollbar styling across Chrome + Firefox
